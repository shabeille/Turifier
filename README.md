A very simple and boring program which generates [turing patterns](https://en.wikipedia.org/wiki/Turing_pattern) from any image via repeatedly blurring and sharpening it.  
[Here](https://archive.bridgesmathart.org/2015/bridges2015-459.pdf) is a pretty cool paper outlining why this works.  
And [here](https://www.youtube.com/watch?v=7oCtDGOSgG8) is a pretty cool video exploring this as well.
